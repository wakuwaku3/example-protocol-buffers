# Usage

## 前提

- docker を install
- protobuf-compiler を install

```bash
apt install -y protobuf-compiler
```

- clang-format を install

```bash
sudo apt install clang-format
```

- go を install
- protolint を install

```bash
go install github.com/yoheimuta/protolint/cmd/protolint@latest
```

## 生成

`make gen`
